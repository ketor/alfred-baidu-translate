# ketor.workflows.baidutranslate

## 百度翻译 workflow v1.0

默认快捷键 `bd`，查看翻译结果。

1. 英译中
2. 中译英
3. 翻译短语句子
4. 打开百度翻译页面，查看详细内容
5. 直接在打出翻译结果

### 功能

1. 按`回车` 复制
2. 按`Control ⌃+回车` 打开百度翻译页面
3. 按`Command ⌘+回车` 直接在打出翻译结果
4. 选中文字 双击`Option ⌥`进行翻译

### 感谢

感谢liszd的插件[https://github.com/liszd/whyliam.workflows.youdao], 基于liszd的代码开发了百度翻译的Alfred插件。
